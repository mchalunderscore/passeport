Integrations

SSH

Configure adds Passeport’s agent to ~/.ssh/config. The authentication key is served by the running app.

OpenPGP

GNU-free installs passeport-gpg. Pluggable Scdaemon connects an existing GnuPG installation to Passeport. The optional gpg alias is installed only when Passeport can do so safely.

Git signing

Choose SSH or OpenPGP signing. Passeport records the global Git values it replaces so Remove can restore them.

age and Minisign

passeport-age decrypts through the app. passeport-minisign signs through the app and verifies publicly. The Minisign public key is written to ~/.minisign/passeport.pub.

Test performs a real private operation followed by verification or decryption. It may request approval according to Settings.
